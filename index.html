pnode

pnode is a Node.js library, built ontop of dnode (What's dnode?). pnode allows server applications to easily communicate over a peer-to-peer network. Since there's no centralised server, there's no single point of failure, this ability simplifies the creation of resilient applications by adding redundancy.
